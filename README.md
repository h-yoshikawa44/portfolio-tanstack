# h-yoshikawa44's Portfolio
h-yoshikawa44 のポートフォリオサイト（リプレース案）

## 構成
- Node.js: 24.13.0
- TypeScript: 5.9.3
- React: 19.2.4
- TanSlack Start: 1.147.3

## 環境構築
パッケージインストール
```bash
npm i
```

環境変数
```bash
cp .env.example .env
```

- VITE_GOOGLE_ANALYTICS_ID: `G-`から始まる Google Analytics の ID
- VITE_GOOGLE_SITE_VERIFICATION: Google Search Console の所有権確認の meta タグの値


サーバ起動
```bash
npm run dev
```
